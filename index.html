HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The abbreviation for HTML means HyperText Markup Language.

CSS

  • The abbreviation CSS stands for Cascading Style Sheets.
  • CSS gives style to the content of the HTML skeleton.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • Github is the site that allows web developers to collaborate!
  • Git commands always start with "git".
  • Git terminology located in seperate section, important to review!
  • The code "git status": checks what branch we are currently on.
  • The code "git checkout -b branch-name": creates a new branch and switches to it

JavaScript

  • The abbreviation JS stands for JavaScript.
  • JavaScript is used to handle logic and allow users to interact with the content.
